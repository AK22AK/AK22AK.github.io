# AK22AK 个人主页

基于 [Astro](https://astro.build) 构建，托管于 [GitHub Pages](https://pages.github.com)。

## 在线访问

🔗 [https://AK22AK.github.io](https://AK22AK.github.io)

## 模块

- **首页** — 个人简介与快速导航
- **关于** — 公开信息与联系方式
- **项目** — 自动同步 GitHub 公开仓库
- **想法** — Markdown 博客文章
- **每日新闻** — 技术资讯聚合

## 本地开发

```bash
npm install
npm run dev
```

## 部署

推送至 `main` 分支后，GitHub Actions 会自动构建并部署。
